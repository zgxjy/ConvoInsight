# 会话质检分析系统

一个现代化的客服会话质检分析系统，专注于用户意图识别、服务解决力和满意度分析。

## 功能特点

- 会话信息与摘要展示
- 关键服务指标分析（满意度、服务质量、主动性）
- 用户意图智能识别
- 服务方案评估
- 服务亮点与改进建议

## 技术栈

- React 18
- TypeScript
- Ant Design 5.0
- CSS3

## 开发环境设置

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm start
```

3. 访问开发环境：
打开浏览器访问 http://localhost:3000

## 项目结构

```
src/
  ├── components/         # React组件
  ├── App.tsx            # 应用入口组件
  ├── App.css            # 全局样式
  └── index.tsx          # 应用入口文件
```
